Passos para clonar um repositório GIT
  • Abra o terminal
  • Escolha um diretório e crie uma pasta com mkdir[nome da pasta]
  • Entre na pasta com cd [nome da pasta]
  • Inicie um repositório dentro da pasta criada com git init
  • No GitHub, copie a url do repositório e clone para seu repositório local com o comando git clone [url]
  • Pronto, você clonou um reposiório remoto para seu repositório local 🗼
